# Ecommerce App (React Native)

A streamlined mobile application built with **React Native** and **Expo**. This project focuses on user authentication and product management, utilizing **Firebase** as a backend solution.

> **Note:** This is a personal project currently under active development.

## 🚀 Features

* **User Authentication:** Secure login system integrated with Firebase.
* **Demo Credentials:**
    * **User:** `usuario@gmail.com`
    * **Password:** `usuario123`
* **Product Management:** Dedicated interface for creating new entries including:
    * Product Name
    * Category/Area
    * Price/Value
    * Image Upload/Attachment

## 🛠 Tech Stack

* **Framework:** [React Native](https://reactnative.dev/)
* **Tooling:** [Expo](https://expo.dev/)
* **Navigation:** [React Navigation](https://reactnavigation.org/)
* **Backend & Auth:** [Firebase](https://firebase.google.com/)
